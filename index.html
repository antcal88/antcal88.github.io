This website includes examples of my work in Ms. Chang's Web Design I class.

Unit 2

In this unit, I relearned the basics of HTML, which I wasn't too familiar with before. I have a better understanding of the materials now.

One of the difficult parts of this unit was getting used to coding once again and getting used to how HTML operates. Some other difficulties were:

    The intolerance of HTML
    Only knowing a small amount of commands
    Finding and correcting mistakes

Unit 3

Unit 3 taught a lot of vital information, info you need to know to truly make your websites look great. With CSS having such a large impact on the appeal of your site, I'm glad we had a lot of time to learn the basics.

Unit 2 prepared me for more coding in Unit 3, so most of the difficulty came from not knowing code or missing small parts. Overall, three of the harder parts of this unit were:

    Mixing around small symbols (such as semicolons and dashes)
    Only knowing a select few commands
    Some CSS just downright not working sometimes

Unit 4

Unit 4 was a lot of fun because it's where we really began to make things looks pretty. I already love anything graphics myself, so being able to impliment that into my work here was a lot of fun for me.

Since I have a lot of past experience, the unit was quite easy for me.

Unit 5

Unit 5 was special to me because it introduced me to another side of Web Design that I've never used before. The coding itself is pretty straightforward, but is also very interesting because it's unique from CSS and HTML. Still, some harder parts of this unit were:

    Unforgiving with spelling/capitalization
    "Interesting" code, might need to look up reference code often
    Easily frustrating with the smallest mistake ruining the entire script

Unit 6

Unit 6 was interesting in the fact that it was the first time we showed our work to other people, mostly from other program areas. It's really an exciting feeling now that I realize we're getting closer and closer to a final professional site. In this mission, we learned:

    How to validate all of our code manually
    How to use online HTML correctors
    How to gather info from a usability test
